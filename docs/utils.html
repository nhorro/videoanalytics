Utilities¶

API reference¶

videoanalytics.utils¶

This module contains miscelaneous functions for conversion of types and adapting format of data shared by more than one component.

videoanalytics.utils.format_boxes(bboxes, image_height, image_width)¶

Helper function to convert bounding boxes from normalized ymin, xmin, ymax, xmax to frame coordinates: xmin, ymin, xmax, ymax

Parameters

  • bboxes (np.array) – Array containing multiple bounding boxes.

  • image_height (int) – height of the image in pixels.

  • image_width (int) – width of the image in pixels.

Returns

converted bounding boxes.

Return type

bboxes (np.array)

videoanalytics.utils.read_class_names(filename)¶

Reads a textfile containing classes names in DarkNet format.

Parameters

filename (str) – text file. See examples.

Returns

A dictionary that maps each class name with an id.

Return type

names (dict)
