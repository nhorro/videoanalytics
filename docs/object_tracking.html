Object tracking¶

Object trackers use information from object detectors to track the position and velocity of unique objects in a scene.

API reference¶

videoanalytics.pipeline.sinks.trackers.sort¶

Implementation of SORT tracking algorithm.

class videoanalytics.pipeline.sinks.trackers.sort.KalmanBoxTracker(bbox)¶

This class represents the internal state of individual tracked objects observed as bbox.

__init__(bbox)¶

Initialises a tracker using initial bounding box.

get_state()¶

Returns the current bounding box estimate.

predict()¶

Advances the state vector and returns the predicted bounding box estimate.

update(bbox)¶

Updates the state vector with observed bbox.

class videoanalytics.pipeline.sinks.trackers.sort.SORT(context, max_age=1, min_hits=3, iou_threshold=0.3)¶
process()¶

This method is called for each active component in the pipeline.

setup()¶

This method is called after all components from the pipeline are instanced.

shutdown()¶

This method is called after the process finished.

videoanalytics.pipeline.sinks.trackers.sort.associate_detections_to_trackers(detections, trackers, iou_threshold=0.3)¶

Assigns detections to tracked object (both represented as bounding boxes) Returns 3 lists of matches, unmatched_detections and unmatched_trackers

videoanalytics.pipeline.sinks.trackers.sort.convert_bbox_to_z(bbox)¶

Takes a bounding box in the form [x1,y1,x2,y2] and returns z in the form [x,y,s,r] where x,y is the centre of the box and s is the scale/area and r is the aspect ratio

videoanalytics.pipeline.sinks.trackers.sort.convert_x_to_bbox(x, score=None)¶

Takes a bounding box in the centre form [x,y,s,r] and returns it in the form [x1,y1,x2,y2] where x1,y1 is the top left and x2,y2 is the bottom right

videoanalytics.pipeline.sinks.trackers.sort.iou_batch(bb_test, bb_gt)¶

From SORT: Computes IOU between two bboxes in the form [x1,y1,x2,y2]
